# *Información general del proyecto*

Nombre del proyecto: technical-test-amaris-frontend.

Propósito: BTG Pactual desea crear una plataforma que permita a los clientes gestionar sus fondos de 
inversio n sin necesidad de contactar a un asesor. 

Tecnologías usadas:

Frontend: React.js con Vite, React-Bootstrap, Axios.

Estilos: Bootstrap + fuente Roboto.

Gestión de rutas: React Router.

Estado global/local: useState, useEffect.
